初涉 GitHub Pages

  • 帮助文档

一时兴起，利用Hexo框架搭建了自己的个人博客，今后会使用git不定时的更新博客的内容。这是我的第一个博客，希望可以利用该博客分享自己的工作经验、个人感悟和随笔。希望得到大家的反馈，相互交流，共同学习！

搭建概要

使用Hexo框架生成静态网站

1.Node.js的安装：百度下载Node.js 64bit .msi
2.安装淘宝npm镜像 npm install -g cnpm --registry=https://registry.npm.taobao.org 
3.安装Hexo框架 cnpm install -g hexo
4.使用Hexo初始化博客项目 首先进入项目部署文件夹，执行命令
  hexo init myblog 
  npm install
5.初始化成功后，进入myblog目录下执行命令：
  hexo s
  可直接访问本地地址：localhost:4000

推送文章网站到GitHub Pages

1. 新建一个仓库（Create a new repository）
  仓库名为：用户名.github.io
2. 成功后返回myblog文件夹，执行命令：
  npm install hexo --save
  hexo generate 
  cnpm install hexo-developer-git --save
3. 完成后文本编辑器打开_config.yml文件，更改文件末尾配置项
deploy:
  type: git
  repository: git@github.com:用户名/用户名.github.io.git
  branch: master（注：' : '后面都要加一个空格，否则会报错）
4. 再执行命令：
  npm install --save hexo-deployer-git
  hexo  deploy
部署完成：https://zhangyang-smart.github.io/

为个人博客的配置与主题设置

1. _config.xml配置文件
 title：网站的标题
 subtitle：网站的副标题
 description：网站的描述
 author：网站的作者
 language：网站使用的语言
 timezone：网站的时区
2.主题设置：
 访问https://hexo.io/themes/
 例如设置主题为 yelee,执行命令：git clone https://github.com/MOxFIVE/hexo-theme-yelee.git themes/yelee
 再修改_config.xml配置文件，更换默认主题为yelee
